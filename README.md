# PesquisaBinaria

Repositório destinado à divulgação da resolução de uma atividade proposta pelo professora Liza na disciplina de Programação orientada a objetos presente no curso de Análise e Desenvolvimento de Sistemas da Uninassau Aracaju
***

A atividade consiste na resolução de uma problemática que exige a implementação da técnica de Pesquisa Binária para otimizar a busca por dados. O objetivo é demonstrar a eficiência desse algoritmo em comparação com a busca linear, especialmente em grandes volumes de dados ordenados.
---

No problema em específico, devemos codificar a solução em C# e aplicar os conhecimentos de Programação Orientada a Objetos (POO) vistos na primeira unidade da disciplina.
